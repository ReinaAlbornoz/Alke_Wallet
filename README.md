# Proyecto Alke Wallet

¡Hola! Este es mi proyecto de Alke Wallet, una aplicación web sencilla que simula una billetera digital. Lo hice como parte de mi curso de Java, pero al final terminé usándolo más para practicar HTML, CSS y JavaScript.

## ¿Qué hace?

La app te permite:
- **Depositar dinero**: Agrega fondos a tu cuenta virtual.
- **Enviar dinero**: Transfiere dinero a otros usuarios (simulado, claro).
- **Ver transacciones**: Revisa el historial de tus movimientos.
- **Menú principal**: Navega fácilmente entre las diferentes secciones.

Todo está hecho con páginas HTML básicas, un poco de CSS para que se vea decente y JavaScript para manejar la lógica. Nada fancy, pero funciona.

## Tecnologías que usé

- **HTML**: Para la estructura de las páginas.
- **CSS**: Para los estilos y que no se vea tan feo.
- **JavaScript**: Para la interactividad y el manejo de datos (todo en local, sin backend real).

## Cómo probarlo

1. Clona el repo.
2. Abre `index.html` en tu navegador favorito.
3. Juega con las funciones. ¡Es gratis y sin riesgos!

Si encuentras algún bug o tienes ideas para mejorar, avísame. Estoy aprendiendo, así que cualquier feedback es bienvenido.
